# appservice

## Checks


- [account-identity-registered](account-identity-registered) Web App has registration with AD enabled

- [authentication-enabled](authentication-enabled) App Service authentication is activated

- [enable-http2](enable-http2) Web App uses the latest HTTP version

- [enforce-https](enforce-https) Ensure the Function App can only be accessed via HTTPS. The default is false.

- [require-client-cert](require-client-cert) Web App accepts incoming client certificate

- [use-secure-tls-policy](use-secure-tls-policy) Web App uses latest TLS version
